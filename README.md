# SafeHaven – AI-Powered Tourist Safety System
 ---
## Description

SafeHaven is an AI-based, cross-platform tourist safety system designed to provide predictive, preventive, and responsive safety assistance in unfamiliar environments.

The system leverages machine learning, Geographic Information Systems (GIS), and real-time data to assess safety risks based on location and time. It enables users to identify high-risk areas, select safer travel routes, and access emergency support when required.

Unlike traditional safety applications that rely on manual SOS activation and shortest-path navigation, SafeHaven integrates predictive analytics to improve situational awareness and enhance user safety.

## Features

- AI-based risk prediction using spatio-temporal data  
- Safe route navigation based on safety levels instead of distance alone  
- QuickShield: discreet SOS activation using hidden triggers  
- SafeSpot: identification of nearby verified safe locations such as cafes, NGOs, and shelters  
- Real-time trip tracking with abnormal activity detection  
- Community-based safety updates and emotional check-ins  
- Dynamic alerts based on changing risk conditions
